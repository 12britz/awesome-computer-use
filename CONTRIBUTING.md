# Contributing to Awesome Computer Use

Thanks for wanting to add to the list! Here is how.

## Adding a Project

1. **Fork** this repo
2. **Find the right category** for your project
3. **Add it** to the table in alphabetical order within that category
4. **Format** — `Project Name` | Stars | One-line description
5. **Submit a PR** with a clear title like "Add YourProject to CategoryName"

## Rules

- **Must be open source** (or have a meaningful open-source component)
- **Must relate to computer use** — desktop automation, GUI agents, screen understanding, etc.
- **Must be actively maintained** — commit within the last 6 months
- **No duplicates** — check if it is already listed
- **No spam** — your project needs to actually work

## What Counts as Computer Use

- Desktop automation agents
- Browser automation with AI
- Mobile device automation
- Screen capture and recording for AI
- OCR and screen reading tools
- Vision models for GUI understanding
- MCP servers for desktop control
- Agent memory systems
- Sandbox and containerization for agents
- Voice-controlled desktop agents
- Game-playing agents
- Training data collection tools

## Star Count

Use approximate star counts. Do not obsess over exact numbers.

## Questions

Open an issue. We are friendly.
